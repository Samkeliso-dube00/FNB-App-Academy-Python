## 🐍 What is Python?

Python is a **high-level, interpreted programming language** that was first released in **1991**.

Its simple and readable syntax makes it one of the most beginner-friendly programming languages while still being powerful enough for professional software development.

Python is widely used in many industries because it allows developers to build applications quickly and efficiently.

## 📄 Your First Python Script

A Python program is stored inside a file with the **`.py`** extension.

Example:

```python
print("Hello, World!")
```

The `print()` function displays information in the terminal.

Running the program produces:

```text
Hello, World!
```

---

## ⌨️ Getting User Input

Python allows users to enter information using the `input()` function.

Example:

```python
name = input("Enter your name: ")

print(name)
```

The value entered by the user is stored inside the variable `name`.

---

## 📦 Variables

A variable is a named location used to store information.

Example:

```python
name = "Samkeliso"
```

Here:

- `name` is the variable.
- `"Samkeliso"` is the value stored inside it.

Variables make it possible to reuse data throughout a program.

---

## 🔤 Python Data Types

Python automatically determines the type of data stored in a variable.

The four basic data types introduced in this unit are:

| Data Type | Description | Example |
|-----------|-------------|---------|
| `str` | Text | `"Python"` |
| `int` | Whole numbers | `25` |
| `float` | Decimal numbers | `3.14` |
| `bool` | True or False values | `True` |

Example:

```python
name = "Samkeliso"
age = 25
height = 1.75
is_student = True
```

---

## 🔍 Inspecting Data Types

Python provides the `type()` function to determine the type of a value.

Example:

```python
name = "Python"

print(type(name))
```

Output:

```text
<class 'str'>
```
