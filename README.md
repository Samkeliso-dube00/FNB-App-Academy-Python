## 🎯 About the Course

The FNB App Academy Python Programming course is a **9-week self-paced program** consisting of six learning units. Each unit builds on the previous one, introducing new concepts through lessons, practical exercises, and quizzes.

My goal is not only to complete the course but also to understand the concepts well enough to apply them in real-world projects.
